# Kala Web - Role & Permissions Management

A modern web interface for assigning roles and managing user permissions in the Kala enterprise system.

## Features

- 👤 **User Profile Management** - View and manage user information
- 🎭 **Role Assignment** - Select from 16 different enterprise roles
- 🔐 **Permission Controls** - Toggle granular permissions with toggle switches
- 📍 **Site & Manager Selection** - Assign users to specific sites and reporting managers
- 🎨 **Modern UI** - Clean, responsive design with smooth animations
- ⚡ **Real-time Feedback** - Toast notifications for user actions

## How to Use

1. Clone this repository: `git clone https://github.com/bhoomijain226/kala-web.git`
2. Open `assign-role.html` in your web browser
3. Select a role from the grid
4. Choose a site and reporting manager
5. Configure permissions using the toggle switches
6. Click "Save Assignment" to save changes
